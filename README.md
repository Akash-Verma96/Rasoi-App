# 🍽️ Rasoi App

## 📌 Overview

Rasoi is a modern full-stack web application designed to provide users with a smooth and interactive food-related experience. The application focuses on user authentication, clean UI/UX, secure routing, and scalable architecture.

The project was built to strengthen practical full-stack development skills using modern web technologies and real-world development practices.

# ⚙️ Installation & Setup

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/rasoi-app.git
```

## 2️⃣ Navigate to Project Folder

```bash
cd rasoi-app
```

## 3️⃣ Install Dependencies

### Frontend

```bash
cd frontend
npm install
```

### Backend

```bash
cd backend
npm install
```

---

# ▶️ Run the Application

## Start Backend Server

```bash
npm run server
```

## Start Frontend

```bash
npm start
```

---

# 🔒 Environment Variables

Create a `.env` file inside the backend folder and add:

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```
